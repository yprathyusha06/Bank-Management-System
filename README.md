# Bank Management System

## Project Description
The Bank Management System is a simple Java console application. It allows users to create an account, log in using an account number and PIN, deposit money, withdraw money, check the account balance, and log out.

## Features
- Create Account
- Login with Account Number and PIN
- Deposit Money
- Withdraw Money
- Check Balance
- Logout
- Exit

## Technologies Used
- Java
- Visual Studio Code
- JDK 24

## Concepts Used
- Classes and Objects
- Variables
- Scanner Class
- if-else Statements
- switch Statement
- while Loop

## How to Run
1. Open the project in Visual Studio Code.
2. Make sure Java JDK is installed.
3. Open `BankManagementSystem.java`.
4. Click the **Run** button.
5. Follow the menu displayed in the terminal.

## Sample Menu
```
====== BANK MANAGEMENT SYSTEM ======

1. Create Account
2. Login
3. Exit
```
